================================================================================

STATISTICS:
-----------
Total Tariff Codes:     9,040
Unique Chapters:        97
Unique Partidas:        1,156
Unique Subpartidas:     5,262

COLUMNS:
--------
1. COD_INCISO (INTEGER, PRIMARY KEY) - Full tariff code
2. COD_CAPITULO (INTEGER, INDEXED) - Chapter (2 digits)
3. COD_PARTIDA (INTEGER, INDEXED) - Heading (4 digits)
4. COD_SUBPARTIDA (INTEGER, INDEXED) - Subheading (6 digits)
5. DESCRIPCIÓN (TEXT) - Official description
6. DAI (TEXT) - Import duty rate
7. ITBMS (REAL) - Tax percentage
8. ISC (REAL) - Selective consumption tax
9. ICCDP (REAL) - Other tax
10. CAPITULO_NOMBRE (TEXT) - Chapter name

================================================================================
JOIN CAPABILITY
================================================================================

PRIMARY JOIN KEY: COD_INCISO

BASIC SYNTAX:
-------------
SELECT * 
FROM merged_imports i
LEFT JOIN catalogo_arancel c ON i.COD_INCISO = c.COD_INCISO

MATCH RATE:
-----------
~78% of import codes match with catalogue
~22% unmatched (may be outdated codes or different formats)

EXAMPLE QUERY - Find all flower imports:
-----------------------------------------
SELECT 
    i.FECHA_IMPORTACION_EXPORTACION,
    i.IMPORTADOR_EXPORTADOR,
    c.DESCRIPCIÓN as producto_oficial,
    i.CANTIDAD,
    i.PRECIO_UNIDAD,
    i.TOTAL_A_PAGAR,
    c.DAI as tarifa_oficial
FROM merged_imports i
INNER JOIN catalogo_arancel c ON i.COD_INCISO = c.COD_INCISO
WHERE c.COD_CAPITULO = 6
